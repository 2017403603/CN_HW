# C++实现HTTP服务器
①　能接收用户浏览器端基于HTTP的GET和POST请求（已实现）
②　能够读取服务器中存储的文件，并返回给请求客户端（已实现）
③　服务器端支持错误处理，如要访问的URL不存在时回复404错误等（已实现）
④　支持文件在服务器上的上传和下载（已实现）
⑤　服务器可以支持压缩，比如gzip。服务器端可以增加压缩支持（已实现）
⑥　支持多线程，来提高处理请求的并发度（已实现）
